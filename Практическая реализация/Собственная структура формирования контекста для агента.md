## Этапы обработки
### Обработка данных
Для поддержания контекста будет использоваться [[Формовщик]]. Он составит контекст на основе истории общения ([[Историк]]), векторизованной базы знаний ([[RAG-система]]), графа смежных понятий, дополнительных источников (будут введены и уточнены позже)
### Передача в модель
Для передачи в модель будет использоваться [[Эндпоинт для модели (chat completions)]].
Структура сообщения для передачи в `messages`:
```json
[
	{
		"role": "system",
		"content": ... # текст основного промпта, включающий в себя:
					   # характер, правила поведения и использования инструментов
	},
	{
		"role": "user",
		"content": "Контекст: {...}" # контектс составленный Формовщиком
	},
	{
		"role": "user",
		"content": [
			{
				"type": "text",
				"text": "Отправитель: {...}\nСообщение: {...}"
			},
			{
				"type": "image_url",
				"image_url": {
					"url": 
				}
			}
		]"" # контектс составленный Формовщиком
	}
]
```
